# Kalkulator Premium

Static frontend untuk kalkulator premium dengan QRIS, request code, dan akses manual.

## Jalankan Lokal
1. Buka folder project.
2. Jalankan lewat Laragon/localhost atau Live Server.
3. Buka `index.html`.

## GitHub Pages
Project ini bisa langsung dipakai di GitHub Pages karena hanya memakai HTML, CSS, JavaScript, dan aset lokal.

Pengaturan Pages:
1. Masuk ke repository GitHub.
2. Buka `Settings` -> `Pages`.
3. Source: `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/root`.
6. Save.

URL akan mengikuti format:
`https://adzipurnomo18.github.io/kalkulator-premium/`

## Catatan Admin
File `admin.html` sengaja tidak ikut commit/push. Simpan hanya untuk admin karena berisi generator kode akses.
